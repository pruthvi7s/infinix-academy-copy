# **App Name**: CareerCraft Navigator

## Core Features:

- Interactive Assessment: Interactive questionnaire to gather user's skills, interests, and values.
- Career Path Suggestions: Generative AI-powered tool to analyze user inputs from the questionnaire and suggest potential career paths, with reasoning about the appropriateness of each path.
- Skill Mapping: Display career paths and required skills in an orderly fashion.
- Personalized Roadmap Generation: Use of generative AI to create personalized learning roadmaps for each suggested career path.
- Exportable Reports: Allows users to export their report as a PDF file for later review or sharing.

## Style Guidelines:

- Primary color: HSL 210, 67%, 53% (RGB hex: #4285F4) to evoke a sense of trust, intelligence, and innovation.
- Background color: HSL 210, 20%, 96% (RGB hex: #F0F4FF), providing a calm and neutral backdrop.
- Accent color: HSL 180, 55%, 40% (RGB hex: #2AB3A7) for highlighting key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective look; suitable for both headlines and body text
- Consistent use of flat, outline-style icons that visually represent different skills, industries, and learning resources.
- Clean, well-organized layout with clear sections for assessment, results, and learning paths to enhance user experience.
- Subtle transition animations and loading indicators to provide feedback and improve engagement.
